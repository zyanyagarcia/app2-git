Informacion sobre el Perfil de Egreso de la Especialidad


La Información que ofrece la carrera de Técnico en Programación permite al egresado, a través de la articulación de saberes de diversos campos, realizar actividades dirigidas a la:

  • Instalacion y desarrollo de software de aplicacion utilizando programación estructurada y orientada objetos en ambientes web y móviles, con alamacenamiento persistente de datos,
  • Así como la configuración y administración de plataforma e-learning y
  • Comercio electrónico.
Durante el proceso de formación de los cinco módulos, el estudiante desarrollará:
Las siguientes competencias profesionales:
  • Desarrolla e instala software de aplicación utilizando programación estructurada, con almacenamiento persistente de los datos
  • Desarrolla software de aplicacion utilizando programación orientada objetos, con alamacenamiento persistente de los datos
  • Ddesarrolla aplicaciones web y móviles
  • Administra sistemas operativos, aplicaciones y servicios
  • Ddesarrolla, administra y configura soluciones de e-learning y comercio electrónico

  • Modulo I

    Desarrolla e instala software de aplicación utilizando programación estructurada, con almacenamiento persistente de los datos

  • Modulo II

    Desarrolla software de aplicación utilizando programacion orientada a objetos. con almacenamineto persistente de los datos

  • Modulo III

    Desarrolla apliucaciones web y moviles

  • Modulo IV

    Administra sistemas operativos, de aplicaciones y servicios

  • Modulo V

    Ddesarrolla, administra y configura soluciones de e-learning y comercio electrónico
